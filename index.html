DICTIONARY

Iterating through a dictionary is a bit different. They have keys and values. We can choose to loop through a dictionary through keys, values, or both.

When you loop over a dictionary without specifying what to loop over, Python iterates through its keys by default.




The loop variable key holds the current key in each iteration.
You can use the key to access its corresponding value using students[key].



  • Using .keys(): You can use .keys() method to loop through all the keys in a dictionary.
    During each iteration, a new key is assigned to the loop variable key and print(key) prints that.


  • Using .values(): You can use .values() method to loop through all the values in a dictionary.
    During each iteration, a new value is assigned to the loop variable value and print(value) prints that.


  • Using .items(): To loop through both keys and values simultaneously, you can use .items() method.
    The .items() method returns a collection of key-value pairs as tuples.
    The for loop unpacks these tuples into two variables -key and value

MEMBERSHIP OPERATOR

Imagine you have a bag full of different chocolates and you want to check if a particular chocolate is present or not without taking everything out. How would you do that?

Well in real life it can be tricky to check if a particular chocolate is present or not but, in Python, we have a quick and easy way to check if something exists within a collection of data. These are called membership operators.

The membership operators are:
  • in: It checks if something exists in a collection.
  • not in: It checks if something does not exist in a collection.
Membership operators work with lists, tuples, sets, dictionaries, and even strings!

Let‚Äôs see them one by one.
